IDEA快捷键

博主之前一直使用Eclipse作为开发工具，正巧换工作也按要求换环境开发，那么就开始接触强大的IDEA了，本文重在记录IDEA提供的快捷键，快捷键以博主mac为标准，如有不同请自行调整。

构建

command+fn+f9 build全部项目
command+fn+shift+f9 build选中项目

搜索&替换

option + Fn + F7 查找类或方法在哪被使用
control + option + H 查看方法调用链
control + H 查看某类的继承结构
shift + shift 搜索
Ctrl+Shift+R 打开替换窗口
alt + Shift + f 搜索任意字符

调试

fn + f8 断点下一步
fn + f7 进方法
fn + shift + f8 下一个断点

代码简写

psvm: 生成main代码块
sout: 生成System.out.println(); 代码块
serr: 生成System.err.println();代码块
iter: 生成foreach代码块

代码编写

Java

command + option + b: 查看当前类的实现类
option + enter: import 相应类
command + option + t: surround with … 生成包围代码，可以生成if 、if else 、 while 、 do while 、for 、try catch等
command + option + o: 查找method
command + o: 覆盖父类某方法，生成override方法
command + n: 生成setter getter
command + option + l: 格式化代码
command + option + O: 删除未import的包
control + fn + enter : 自动生成构造器方法（control + fn相当于windows键盘的insert）

其他

alt + shift + u. 选中的小写转成大写

替换

使用快捷键Ctrl+Shift+F打开搜索窗口，或者通过点击Edit–>Find–>Find in path打开搜索窗口
搜索具体关键字再做Replace
